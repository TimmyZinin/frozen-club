# Контент-план Frozen Club — 12 недель

## Принципы

- **2 новых мини-урока/месяц** — разборы случаев из практики Кати
- **1 живой вебинар/месяц** — для тарифа Профессионал+ (запись для всех)
- **1 подборка/месяц** — новинки индустрии, полезные статьи
- **1 «Разбор от Кати»/месяц** — видео-ответы на вопросы учениц
- **Еженедельный дайджест** — email каждый понедельник

## Шаблон еженедельного дайджеста (email)

**Тема письма:** «Frozen Club: что нового на этой неделе»

**Структура:**
1. Новый урок / вебинар (если есть) — ссылка
2. Лучший вопрос недели из чата + ответ Кати
3. Полезная ссылка / статья / инструмент
4. Мотивационная цитата или мини-история из практики
5. Напоминание: «Ты прошла X% курса [Название] — продолжай!»

## Планирование контента на следующий квартал

Контент планируется в последнюю неделю текущего квартала:
1. Катя выбирает темы мини-уроков на основе запросов из чата
2. Вебинары привязываются к актуальным темам (сезонность, новинки)
3. Подборки готовятся заранее из накопленных материалов
4. Разборы формируются из вопросов учениц
