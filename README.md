# Frozen Club — Закрытый клуб мастеров педикюра Кати Басковой

## О проекте

Собственная LMS-платформа для закрытого образовательного клуба Екатерины Басковой (@frozen.nail). Подписочная модель с ежемесячной оплатой, видеоуроки, сообщество, обратная связь от Кати.

**Домен:** learn.baskova.pro
**Клиент:** Екатерина Баскова — блогер, автор курсов, преподаватель педикюра и подологии (8+ лет опыта, 10 000+ учеников)

## Стек

- **Backend:** Python 3.12 + FastAPI + SQLAlchemy 2.0 + aiosqlite
- **Frontend:** Vanilla HTML/CSS/JS (mobile-first SPA)
- **Database:** SQLite (MVP) → PostgreSQL
- **Payments:** Prodamus (рекурентные, рассрочки)
- **Video CDN:** Bunny.net (signed URLs)
- **Email:** Unisender
- **Hosting:** Российский VPS (Selectel/Timeweb)

## Курсы в клубе

1. **Титановая нить** — установка титановой нити при вросшем ногте (флагман)
2. **Дисковый педикюр** — скоростная дисковая техника обработки стоп
3. **Секcиногтусы 1.0** — работа с деформированными и вросшими ногтями
4. **Лаковое покрытие в педикюре** — особенности покрытия + однофазные гель-лаки
5. **Мозоли и ВПЧ** — диагностика и методы обработки
6. **Бизнес для мастера педикюра** — позиционирование, ценообразование, привлечение клиентов

## Запуск

```bash
# Backend
cd backend
pip install -r requirements.txt
cp .env.example .env
# Отредактировать .env
python -m uvicorn main:app --host 0.0.0.0 --port 8000

# Docker
docker-compose up -d
```
